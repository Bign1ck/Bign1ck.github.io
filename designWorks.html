Movie Website Overview

This Movie Review Website is made using HTML, CSS and PHP.

Welcome to my Movie Website which I created for my Graded Assignment, and also the first website that I created. It is suppose to mimic closely to movie websites that exist today such as Shaw Theatres or Golden Village. It has a user-friendly interface so that users can navigate through easily. They are 4 main areas, the Home Page, Movie List, Login or Register Page and the More Info Page. Through this project, it has allowed me to learn how to combine the use of PHP, HTML and CSS to create a basic functioning website that would help me with my Website creations in the future

Features include:

1. Homepage that shows movies that are coming soon, that has a footer for customer support

2. Register and Login page that allows new users to sign up and login

3. Movie list page that shows all the movies that are screening now

4. A info page that shows the users more info about the movie as well as looking or even writing the reviews if they are logged in

5. A phpMyAdmin database that allows me to store all the information I need such as the emails, password, name , address, etc of users.
